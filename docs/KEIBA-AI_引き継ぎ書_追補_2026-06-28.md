## システム全体像

```
スマホアプリ（index.html）
    ↑ latest.json / stats.json
GitHub Actions（金/土/日ワークフロー）
    ↓ main checkout → 実行
scripts/friday_predict.py   ← 土曜レース予想生成
scripts/weekend.py          ← 土曜結果取得 + 日曜予想生成
scripts/sunday_results.py   ← 日曜結果取得・DB保存・ROI集計
    ↓ 予測エンジン
src/features/engine.py      ← 特徴量計算（XGB + ルールベース）
src/models/predict.py       ← softmax / キャリブレーション
src/betting/make_bets.py    ← 旧買い目生成（ルールベース）
src/betting/app_json.py     ← latest.json 生成（gumbel_bets も含む）
    ↓ 新買い目エンジン（2026-07-02〜）
src/betting/bet_optimizer.py   ← EV買い目生成（Gumbel確率ベース）
src/betting/dual_model.py      ← B2_ndcg（単勝）+ A_fukusho（複勝等）
src/betting/race_simulator.py  ← Gumbel-Max シミュレーション
src/betting/payout_estimator.py ← 推定配当計算
```

## Colabでの主要操作

### B2モデル再学習（データ蓄積後に実施）

```python
# KEIBA_チューニング_v1.ipynb のセル1直後に追加

import urllib.request, os, sys
BASE_URL = 'https://raw.githubusercontent.com/hanagenuku/keiba_ai/main'
for rel in ['src/tools/train_ranking_model.py', 'src/betting/rating_calibration.py',
            'src/betting/race_simulator.py']:
    dest = f'{BASE_DIR}/{rel}'
    os.makedirs(os.path.dirname(dest), exist_ok=True)
    urllib.request.urlretrieve(f'{BASE_URL}/{rel}', dest)
for key in list(sys.modules.keys()):
    if 'src' in key: del sys.modules[key]

# B2 学習
from src.tools.train_ranking_model import train_ranking_model
train_ranking_model(BASE_DIR, objective='rank:ndcg', model_suffix='ndcg')

# 温度校正
from src.betting.rating_calibration import calibrate_all_models
calibrate_all_models(BASE_DIR, val_start='2026-04-01', val_end='2026-05-31')
```

### モデルファイルをGitHubへpush（Drive→GitHub）

```python
from google.colab import userdata
import subprocess, shutil

PAT = userdata.get('GITHUB_PAT')
REPO = '/content/keiba_ai_push'
subprocess.run(f'git clone https://{PAT}@github.com/hanagenuku/keiba_ai.git {REPO}', shell=True)
for f in ['xgb_ranking_ndcg.pkl', 'xgb_ranking_feature_cols.json', 'rating_temperature.json']:
    shutil.copy(f'{BASE_DIR}/data/{f}', f'{REPO}/data/{f}')
cmds = [
    f'git -C {REPO} config user.email "bot@keiba_ai"',
    f'git -C {REPO} config user.name "keiba_ai bot"',
    f'git -C {REPO} add data/xgb_ranking_ndcg.pkl data/xgb_ranking_feature_cols.json data/rating_temperature.json',
    f'git -C {REPO} commit -m "Update B2 model and temperature calibration"',
    f'git -C {REPO} push origin main',
]
for cmd in cmds:
    r = subprocess.run(cmd, shell=True, capture_output=True, text=True)
    print(r.stdout or r.stderr)
```
