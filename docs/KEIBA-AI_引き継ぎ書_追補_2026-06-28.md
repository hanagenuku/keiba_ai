## システム全体像

```
スマホアプリ（index.html）
    ↑ latest.json / stats.json
GitHub Actions（金/土/日ワークフロー）
    ↓ main checkout → 実行

scripts/weekend.py          ← 土曜結果取得 + 日曜予想生成

    ↓ 予測エンジン
src/features/engine.py      ← 特徴量計算（XGB + ルールベース）
src/models/predict.py       ← softmax / キャリブレーション
src/betting/make_bets.py    ← 旧買い目生成（ルールベース）
src/betting/app_json.py     ← latest.json 生成（gumbel_bets も含む）
    ↓ 新買い目エンジン（2026-07-02〜）
src/betting/bet_optimizer.py   ← EV買い目生成（Gumbel確率ベース）
src/betting/dual_model.py      ← B2_ndcg（単勝）+ A_fukusho（複勝等）
src/betting/race_simulator.py  ← Gumbel-Max シミュレーション
src/betting/payout_estimator.py ← 推定配当計算
```
